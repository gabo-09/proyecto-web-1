    CONTENIDO


    -Para qué sirven las siguientes etiquetas?
    Meta : sirve Para aportar información sobre el documento
    Head : para información sobre el documento
    Tittle : por escribir el título
    !DOCTYPE html : para definir el tipo de documento
    Body : para el contenido
    a : para enlaces
    Table : para dar un conjunto de celdas
    Caption : para darle un título descriptivo a las tablas
    Th : para definir una celda como encabezado
    Td : para definir la celda de una tabla que tiene datos
    Tr : para definir una fila de celdas en una tabla
    h1 : para títulos dentro del contenido
    Hgroup : para representar el encabezado de una sección
    p : para párrafos
    Img : para Añadir imágenes al documento

    La etiqueta -a- cuenta con valores , para qué sirven cada uno?
    _self muestra el documento enlazado en la misma ventana del enlace
    _blank abre el link del documento en una nueva ventana
    _parent provoca que el documento se muestra en la parte actual
    _top muestra en la ventana completa , destruyendo el reframes

    Para qué sirven los siguientes atributos de meta?
    Lang: especifica el idioma del texto que contiene un elemento
    Charset: indica la codificación de carácteres utilizados
    Name: permite Acceder al contenido
    Content: genera nuevo contenido de forma dinámica para insertar en la página html
